# 🤖 FAQBot AI

An AI-powered FAQ Chatbot built using **Python, Flask, NLTK, Scikit-learn, HTML, CSS, and JavaScript**.

## 🚀 Features

- 💬 AI FAQ Chatbot
- 🔍 NLP Text Preprocessing
- 📊 TF-IDF + Cosine Similarity
- 🎨 Modern Responsive UI
- ⚡ Real-Time Chat
- 💾 Chat History
- 🧠 Intelligent FAQ Matching
- 📱 Mobile Friendly

---

## 🛠️ Tech Stack

- Python
- Flask
- NLTK
- Scikit-learn
- HTML5
- CSS3
- JavaScript

## ⚙️ Installation

```bash
git clone https://github.com/ghirdharbhadada-arch/FAQBot-AI.git

cd FAQBot-AI

pip install -r requirements.txt

py app.py
```

Open:

```
http://127.0.0.1:5500/LinguaX-AI/index.html
```
